A framework for uncertainty quantification and test-time exploration. 

The main object of the repository is the Process Uncertainty Model (PUM), which can be trained to quantify uncertainty accross steps of a Chain of Thought (CoT).

One specific test-time use case of PUMs is Step-Level Beam Search (SBS), here augmented with dynamic budget allocation and exploration scores.
